O GIT HUB

E uma plataforma de hospedagem de codigo-fonte e arquivos com controle de versão usando o git. Ele é amplamente utilizado por desenvolvedores para colaborar em projetos de software, permitindo que várias pessoas trabalhem no mesmo código-fonte simultaneamente.
